# IT
### Vibe Coding
- **바이브 코딩(Vibe Coding)**은 2025년 2월 OpenAI 공동 창립자인 Andrej Karpathy가 처음 제시한 개념으로, 개발자가 AI 서비스의 도움을 받아 자연어로 코드를 작성하는 새로운 프로그래밍 방식을 의미
- 사전에 엄밀한 논리나 설계보다는 직감과 느낌에 의존하여 코드를 생성한다고 하여 '바이브' 코딩이라는 이름이 붙음
### MCP(Model Context Protocol)
- **MCP(Model Context Protocol)**는 2024년 11월 Anthropic이 도입한 개방형 표준 프로토콜로, AI 모델이 외부 데이터 소스 및 도구와 상호작용하고, 문맥을 더 정확하게 이해하며, 상황에 맞는 작업을 수행할 수 있도록 돕는 기술
### Netlify
- **Netlify**는 웹 개발자를 위한 클라우드 기반의 플랫폼으로, 웹사이트와 웹 애플리케이션의 배포, 호스팅, CI/CD(지속적 통합 및 배포), 서버리스 기능 등을 통합적으로 제공
- 특히 정적 사이트 배포에 최적화되어 있으며, JAMstack(JavaScript, API, Markup) 아키텍처를 기반으로 한 프로젝트에 강력한 기능을 제공
### Gemini CLI
- **Gemini CLI**는 구글의 Gemini AI 모델을 터미널에서 직접 활용하도록 설계된 명령줄 인터페이스 도구
- 텍스트 생성, 코드 작성, 아이디어 구상 등 다양한 AI 작업을 터미널 명령어 입력만으로 빠르고 효율적으로 수행할 수 있음
